CiteBox by willkurt

About

CiteBox is a tool that maks it easy for library users to simply paste citations that they have and, using Brown Universities FreeCite API (http://freecite.library.brown.edu/), parse their citation and redirect them to there libraries link resolver. The project is open source, and designed to be as easy to deploy as possible.

Example

Below is a live example of the CiteBox widget in action. For this particular example I've used The University Of Nevada, Reno's link resolver so you probably cannot get to the actual citations, however you can still look at the generated OpenURL and get a sense of how CiteBox can easily integrate with existing webpages.

Getting Started...

I've tried to make getting started with CiteBox as easy as possilbe. If you decide you want to use CiteBox as a full-time service you should make sure to install your own instance (details on that coming soon). However to get started all you need to do is create a 'div' to hold you CiteBox and replace {my-openurl-base} with the appropriate url. (make sure you are using jquery) 

<link rel="stylesheet" type="text/css" 
      href="http://cite-box.heroku.com/css/citebox.css"></link>
<script type="text/javascript" 
src="http://cite-box.heroku.com/citebox/cite-box.heroku.com/{my-open-url-base}">
</script>
<script type="text/javascript">
  $('document').ready(function(){
      CiteBox.attach($("#my_div"));
  });
</script>
